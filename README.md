# Yeezy's Thoughts

## About

A react app using a random Kanye West quotes api
https://kanye.rest/

View live at
[Yeezy's Thoughts](https://www.yeezysthoughts.com/)

## Install

### Clone the repository

```
git clone https://github.com/MetkelK/yeezysthoughts.git

cd yeezysthoughts

npm install

```

### Start locally

```
npm start
```
